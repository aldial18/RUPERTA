¡Bienvenido al proyecto de Página Web de ruperta Diseño de Interiores! En este archivo README, encontrarás información esencial sobre el proyecto y sus características.

Descripción del Proyecto aún en desarrollo:

Este proyecto tiene como objetivo crear un sitio web dedicado al diseño de interiores. Se han realizado varias mejoras y agregados notables:

Animación en el Logo de la Marca: Se ha añadido una animación al logo de la marca en la página de inicio (index.html) para darle un toque de dinamismo y estilo.

Uso de SCSS: El proyecto ahora utiliza SCSS (Sass) en lugar de CSS convencional. Esto permite una mejor organización del código, la creación de variables para colores y estilos, y la generación de anidaciones para un codigo más ordenado.

Mejora de SEO: Se han implementado etiquetas de meta description en todas las páginas del sitio web para mejorar su visibilidad en los motores de búsqueda. Esto ayudará a aumentar la exposición y la accesibilidad del sitio.

Diseño Responsive: Se ha completado la adaptación de todas las páginas del sitio web para garantizar que se vean y funcionen de manera óptima en una variedad de dispositivos, incluyendo computadoras de escritorio, tabletas y teléfonos móviles.
